# UltimateKEYS Keyboard Layout (Windows/Linux)
International US QWERTY layout with AltGr (Right Alt) dead keys via AutoHotkey script (Win), MSKLC source (Win) and XKB integration (Linux). Derived from EurKEY (by Steffen Br&uuml;ntjen).
