UltimateKEYS Keyboard Layout - MSKLC (Windows)

Integration in Windows OS using MSKLC (not recommended on Win 10 and/or in a production environment)

Important - Please note :

  • I recommend using the AutoHotkey script as it is a less radical approach to the Windows OS.
  • I do not recommend deploying the MSKLC version in a production environment.
    This because Microsoft Keyboard Layout Creator 1.4 is older software (published on 2007-01-30) which has not yet been revised for upstream compatibility and integration in newer Windows versions.
  • Alternatively, KbdEdit can be used for opening and using MSKLC source files.
  • The MSKLC source files can be found in the 'msklc' folder.

 

Tools for opening and handling MSKLC files

  • Microsoft Keyboard Layout Creator 1.4 :  https://www.microsoft.com/en-us/download/details.aspx?id=22339
  • KbdEdit :  http://www.kbdedit.com/
